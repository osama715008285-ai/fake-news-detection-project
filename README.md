\# Fake News Detection System



This project is a Machine Learning web application that detects whether a news text is Fake or Real.



\## Technologies Used



\- Python

\- Streamlit

\- Machine Learning

\- Logistic Regression

\- TF-IDF

\- Joblib



\## How to Run



pip install -r requirements.txt



streamlit run app.py
